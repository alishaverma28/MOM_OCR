# AI-Powered Minutes of Meeting (MoM) Generator

An AI-assisted web application that automatically generates professional and standardized **Minutes of Meeting (MoM)** from meeting notes uploaded in **PDF, DOCX, or Image** formats. Built using **Streamlit**, **Python**, and **Google Gemini Generative AI**.
